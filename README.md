# Data-Analysis

Data Analytics Projects Portfolio
This repository showcases end-to-end data analytics workflows, demonstrating practical skills in transforming raw data into actionable business insights using Python → PostgreSQL → Power BI integration.

Technology Stack
Data Processing: Python (pandas, numpy, matplotlib, seaborn, plotly)

Database: PostgreSQL for data storage and querying

Visualization: Power BI (DAX for interactive dashboards)

Specialized Tools: FastF1 for racing analytics

Getting Started
Each project folder contains:

Python Notebooks/scripts for processing & analysis

PostgreSQL schema and SQL query files

Power BI dashboard files (.pbix)

To run a project:

Open the Python notebook to review/execute data preparation and analysis.

Use SQL for advanced queries or metrics.

Launch Power BI dashboards for interactive exploration.
